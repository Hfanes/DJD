# Unity-Game
Unity RPG Adventure Game - Third Person and Dark Souls style 
3 levels + Boss level
Key features:
Health and Stamina Bar
Movement - Attack, Rolling, Sprint
3 Different Swords with different damages
Lootable Chests (Pick up swords and keys)
Boss Fight
